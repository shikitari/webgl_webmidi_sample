# WebGL Webmidi sample

## Getting Started
### Install node.js
- https://nodejs.org/en/

### Install node packages
```
npm install

npm run build
```
open 'public/index.html' on Chrome.

## system requirements on build
- macOS + Chrome (Latest)
- Windows10 with Ubuntu + Chrome (Latest)
